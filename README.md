# Watchmate

Watchmate is a Movies Info API made with Django REST Framework and Python.

## Features

- JSON Web Token (JWT)

- Token

- Throttling

- Pagination

- Testing

## Technologies

- Built with Django Rest Framework and Python.

- Uses a relational database (e.g: MySQL or PostgreSQL)
